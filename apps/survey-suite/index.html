Survey Suite Workspace

Flujo recomendado: Importa y limpia datos, define la escala Likert y finalmente visualiza resultados.

1. Data Processor

Importa archivos CSV/JSON/XLSX, limpia datos y transforma columnas para análisis.

2. Likert Charts

Crea gráficos Likert, ordena ítems y ajusta estilos para reportes docentes.

3. Distribution Lab

Analiza distribución con boxplot, violin y raincloud para validar resultados.

  1. Carga o selecciona el dataset activo en la cinta superior.
  2. Prepara los datos en Data Processor y guarda en Storage.
  3. Abre Likert Charts o Distribution Lab para visualizar y exportar.
