Ayuda de Exe Builder XAI

Guía práctica para crear, explicar (XAI), editar y exportar ejercicios para docencia y para el visor interactivo.

Flujo recomendado

  1. Ingresa tu API Key y selecciona modelo.
  2. Indica la cantidad de ejercicios (sugerido para pruebas: 3).
  3. Pega contenido o sube un resumen (.txt, .md, .json, .docx, .pdf).
  4. Genera con IA y revisa el Explorador de ejercicios (lista + detalle).
  5. Edita contenido, interacción y campos XAI.
  6. Valida y exporta según el objetivo (proyecto docente o visor).

¿Qué retorno XAI entrega el sistema?

Además del ejercicio en sí, cada ítem incorpora un bloque xai con justificación pedagógica y trazabilidad.

why_this_exercise pedagogical_alignment content_selection design_rationale fairness_and_risk human_oversight quality_of_explanation uncertainty counterfactual trace
  • pedagogical_alignment: objetivo, competencia, Bloom y dificultad del ejercicio.
  • content_selection: por qué se eligió ese contenido y qué fuentes lo sostienen.
  • fairness_and_risk + uncertainty: riesgos/sesgos y límites del resultado IA.
  • human_oversight: protocolo de revisión docente y decisión final bajo supervisión humana.
  • quality_of_explanation: calidad de la explicación según audiencia, claridad y accionabilidad.
  • counterfactual: qué cambiaría si cambia la condición del grupo/escenario.
  • trace: rastro de generación (modelo, prompt, timestamp).

Esto permite auditar decisiones didácticas, no solo consumir preguntas.

Edición dentro de la app

  • Panel izquierdo: configuración y acciones globales.
  • Panel derecho: lista de ejercicios + detalle del ejercicio seleccionado.
  • Sección Interacción: aquí editas opciones/respuestas (según tipo).
  • Sección Vista previa: muestra cómo se verá el ejercicio al estudiante.
  • Los paneles son redimensionables para trabajar con mayor comodidad.

Dos salidas: docente y visor

La app ofrece dos exportaciones para cubrir trabajo continuo y publicación:

  • Exportar proyecto docente: guarda el JSON completo (incluye bloque XAI y metadatos de edición). Sirve para retomar trabajo después.
  • Exportar para visor: genera un JSON simplificado compatible con apps/ejevisor (sin bloque XAI interno).

Vínculo con el visor: diseñas y explicas en Exe Builder XAI, y publicas/ejecutas en EjeVisor con el paquete de visor.

Continuidad del trabajo docente

  • Usa Cargar proyecto docente para recuperar una sesión guardada.
  • Actualiza ejercicios y justificaciones XAI cuando cambie el curso o el contexto.
  • Reexporta para visor cada vez que publiques una versión nueva para estudiantes.

Problemas comunes

  • PDF sin texto: probablemente es escaneado. Conviene OCR previo.
  • Error de validación XAI: revisa campos mínimos (fuentes, confianza, contrafactual).
  • No veo opciones: edítalas en sección Interacción del ejercicio.
