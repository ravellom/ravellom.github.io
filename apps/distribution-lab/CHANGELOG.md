# CHANGELOG

All notable changes to Distribution Lab are documented in this file.

## [0.6.0-alpha] - 2026-03-06

### Added
- Multi-select numeric variables without category for same-chart comparisons.
- Toggle to show/hide sample size in labels (`n=...`).
- Style tabs aligned with Likert-style grouping (Theme, Marks, Layout, Notes).
- Export workflow aligned with Likert (Format + Scale + Download).
- DPI selector for raster exports (96/150/300/400).
- Toggle to show/hide chart title.

### Changed
- Title rendering now respects hidden title state in all chart renderers.
- SVG vector export now respects hidden title state.
